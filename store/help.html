Comment Y Jouer ?

Voici un petit guide, pour aider les gens qui n'arrivent pas à jouer au jeu.

Version Python :

Tout d'abord, vous devez d'abord installer python3

Comment installer python3 sur son ordinateur ?

linux :

Pour installer python3 sur votre ordinateur linux, vous pouvez l'installer depuis le site de python

- Lien vers la page de téléchargement de python3 pour linux

Ou tout simplement lancer la commande :

sudo apt-get install python3 python3-pip
Windows :

Pour installer python3 sur votre ordinateur Windows 10, vous devez aller sur le site de python3

- Lien vers la page de téléchargement de python3 pour windows

Vous devez télécharger la dernière version qui est compatible avec votre ordinateur

Vous installez le fichier téléchargé

Attention, n'oubliez pas de cocher la case "Add Python 3.x to PATH" lors de l'installation de python3, sinon vous ne pourrez pas faire la suite !

Après, vous devez installer les librairies de python3 que le jeu utilise

Comment installer les librairies de python3 sur son ordinateur ?

linux et windows :

C'est très simple, vous devez juste aller sur votre terminal, invite de commande

N'importe quel terminal avec les commandes bash système pour Linux, CMD pour windows

Après, le difficile est passé, vous n'avez qu'à lancer la commande :

sudo pip3 ou pip install Les dépendances qui sont marquées au dessus de la page

ou

pip3 ou pip install Les dépendances qui sont marquées au dessus de la page --user

Normalement, Vous avez installé python3 et les librairies nécessaires sur votre ordinateur

Il ne vous reste plus qu'une chose : lancer le jeu et jouer !

Comment le jeu avec python3 sur son ordinateur ?

Pour le lancer, il y a plusieurs manières disponibles.

La plus simple, utiliser un script bash/bat pour lancer le jeu :

linux :

Vous pouvez télécharger ce script

Et vous avez juste à le modifier si il ne marche pas ( ce n'est pas très compliqué, vous n'avez qu'à suivre les commentaires que j'ai écrits dans les petits scripts ).

Et puis vous n'avez plus qu'à executer le script pour lancer le jeu

Attention, n'oubliez pas de mettre le script dans le meme dossier que le jeu, sinon, il ne marchera pas !

Vous pourrez quand meme juste en faire un raccourci pour le mettre dans votre bureau si vous en avez envie

windows :

Vous pouvez télécharger ce script

Et vous avez juste à le modifier si il ne marche pas ( ce n'est pas très compliqué, vous n'avez qu'à suivre les commentaires que j'ai écrits dans les petits scripts ).

Et puis vous n'avez plus qu'à executer le script pour lancer le jeu

Attention, n'oubliez pas de mettre le script dans le meme dossier que le jeu, sinon, il ne marchera pas !

Vous pourrez quand meme juste en faire un raccourci pour le mettre dans votre bureau si vous en avez envie

Une un peu plus compliquée, utiliser le terminal/invite de commande pour lancer le jeu :

Il n'y a que 2 commandes à éxécuter, sur linux ou windows :

cd répertoire du jeu ( ex : /home/user/Download/jeu/ )

Et

python3/python/py nom du jeu ( souvent : main.py )

Version Windows Executable :

Ce n'est pas très compliqué, pour pouvoir jouer au jeu, il suffit de télécharger le fichier compressé

Ensuite, il faut l'extraire, pour cela, vous pouvez utiliser votre gestionnaire de fichier, click doirt sur le fichier, et cliquer sur extraire.

Après avoir extrait le fichier, il suffit de se déplacer dans le dossier qui résulte de l'extraction précédente, puis lancer le fichier executable

En général, ce fichier se nome : "main.exe" , "jeu.exe" ...
